

## Build
```
npm run build
```

или
```
./node_modules/.bin/webpack
```


## Watch
```
npm run watch
```
или
```
./node_modules/.bin/webpack -w
```


## Watch and autodeploy to postgres
```
npm run watch -- --env.host=192.168.103.7
```
или
```
./node_modules/.bin/webpack -w --env.host=192.168.103.7
```
